# x (for terminal)

<img src="img/x-logo-folderguy.jpg" alt="x" width="28%" height="28%">

CD into any dir simultaneously during Make command.
Save time by not having to cd 'intomydir'

mkdir myfoldername;
cd myfoldername;
becomes =>
x mkdir myfoldername

rails new myapp;
cd myapp;
becomes =>
x rails new my app
